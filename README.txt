音乐可视化器
=============

屏幕底部彩虹频谱条 + 边缘光带，实时捕获系统音频。

使用方法:
  1. 先装 Python 3.11+（需勾选 Add to PATH）
  2. 双击 install.bat 安装依赖
  3. 双击 start.bat 启动
  4. 系统托盘有图标，右键可设置或退出

功能:
  - 彩虹频谱条（80 根，随音乐跳动）
  - 边缘光带（从底部向上，颜色随频率变化）
  - 设置面板（可调条数、颜色、灵敏度等）
  - 开机自启（默认开启）

需要:
  - Windows 10/11
  - Python 3.11+
  - 声卡（会捕获系统播放的声音）
